Python made a way as framework to use python in web development. There are several number of framework to use But, the Django is quite famous than other things and flask framework is also good.

What is Django?

Framework is build to make our website quickly as well as user friendly. Django is a framework build with Python and it’s not a front-end and back-end. It is a collaboration of both.

We can build a massive website in just a months or less with Django. So, it is widely used in web development when we decide to use python for our website.

Install Django in Windows

We can’t install Django like other softwares. We have to follow certain procedure to install Django in your local system. Before install Django, make sure you have python in your system.

  • If you want Django in system, then open windows terminal and paste this pip install django and check Django is installation with this commend django-admin - -version. 
    pip install django
django-admin --version
